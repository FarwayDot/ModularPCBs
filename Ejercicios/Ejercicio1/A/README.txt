************************* EJERCICIO 1 - A *********************************

Con un pulsador y  2 leds, implementar una secuencia en binario de 0 a 3
cada vez que se presiona el pulsador. Configurar el pin 3 del puerto B para
la entrada del pulsador. Configurar el pin 2 del puerto B y el pin 4 del 
puerto A para la salida de los leds.
